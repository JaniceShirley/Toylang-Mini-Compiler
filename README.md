# ToyLang Mini Compiler

ToyLang is a small custom programming language built in Python.

This project demonstrates the basic compiler pipeline.

## Compiler Pipeline

ToyLang Code
↓
Lexer (Tokenization)
↓
Parser (Builds Abstract Syntax Tree)
↓
Code Generator
↓
Python Code Execution

## Example ToyLang Code

let a = 10
let b = 5
print a + b

## Output

15

## Features

- Variable assignment
- Arithmetic operations (+ - * /)
- Print statements
- ToyLang → Python code generation
